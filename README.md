# Udacity - Restaurant Reviews App
---
### Instructions
-
#### Start
* git clone https://github.com/ImranK1506/Udacity-RestaurantReviews
* NPM run start

#### Stop server
* Stop server: ctrl + c

_In case of server problems follow instructions underneath:_

* List all current connections: ps -ef | grep python
* Kill server: kill *id
